# ecommerce-Sales-Delivery-and-Customers-Satisfaction-Analysis
## About This Project
In this project, I analyzed an e-commerce dataset to understand sales performance, delivery time, and customer satisfaction.
I used SQL, Python, and Power BI to go from raw data to a final dashboard.

# Tools Used
-SQL
-Python (Pandas, Matplotlib, Seaborn)
-Power BI

# What I Did
-Cleaned and prepared data using SQL
-Explored the data using Python
-Created a dashboard in Power BI

# Dashboard Includes
-Total Revenue and Orders
-Average Delivery Time
-Customer Ratings
-Sales Trend by Month
-Payment Type Distribution
-Top Products

# Key Insights
-Late deliveries lead to lower customer ratings
-Most customers use credit cards (~70%)
-Sales increased in 2017 and then started to decline
-Average rating is around 4
